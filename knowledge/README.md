# Knowledge Layout

Use this flow:

1. Drop raw files in `knowledge\inbox`
2. Clean and rename files clearly
3. Move to one domain folder:
   - `knowledge\business`
   - `knowledge\coding`
   - `knowledge\personal`
4. Upload each domain folder into its own Open WebUI knowledge space

Guidelines:

- Prefer markdown, pdf, txt, csv
- Keep one topic per file
- Include date in filename for time-sensitive docs
